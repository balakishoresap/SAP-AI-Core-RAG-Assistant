Frontend Layer

Single-page Fiori Horizon UI at /app/webapp/. Uses vanilla JS with SAP Horizon CSS variables. Includes a demo mode with 10 pre-built answers when backend is unavailable (e.g., GitHub Pages).

Fiori Horizon Theme

CAP Service Layer

AskService extends cds.ApplicationService. Registers on('ask') handler that orchestrates the RAG pipeline. Persistence is non-blocking via _persistAsync().

@sap/cds v8+

AI Pipeline

6-step RAG process: Pre-process → Embed → Search → Prompt → LLM → Post-process. Demo uses character-frequency embeddings; production uses SAP AI Core embedding models.

384-dimensional vectors

Data Layer

CDS-modeled entities deployed to SQLite (dev) or SAP HANA Cloud (prod). 4 entities: KnowledgeBase, Questions, Answers, AuditLog.

@cap-js/sqlite
